Test using Arduino and a transistor

Jan 19, 2022

The MT-240 has a keyboard. And each key is a momentary switch (a button) that corresponds to a spot in a keyboard matrix [link]. So when a key is pressed...an electrical connection is made between a row and column in the matrix, which tells the CPU which note to play. For example, pressing the 'A4' key corresponds to row 4, col 11 on the MT-240.

But we can use other types of switches too! Like a transistor. So I connected a Arduino to my computer to supply power to it, and found a simple code & breadboard setup online.

Instead of the LED, I connected wires from the Collector and Emitter pins that led to the keyboard matrix. It worked! With a time delay of 1000ms, it alternatingly played & stopped a note every second.
